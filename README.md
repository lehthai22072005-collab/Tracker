# 💰 Expense Tracker System

## 📖 Giới thiệu
Hệ thống quản lý chi tiêu cá nhân giúp người dùng theo dõi các khoản thu và chi hằng ngày, từ đó kiểm soát tài chính hiệu quả hơn.

---

## 🛠 Công nghệ
- Java Spring Boot (Backend)
- ReactJS + Vite (Frontend)
- RESTful API
- (MySQL / H2 Database)

---

## 🚀 Chức năng
- CRUD giao dịch (thêm / sửa / xóa)
- Phân loại thu / chi
- Hiển thị danh sách giao dịch
- Tính tổng thu và chi

---

## ⚙️ Cài đặt

### 1. Clone project
git clone https://github.com/your-username/expense-tracker.git  
cd Tracker-main  

### 2. Chạy Backend
cd QLCT-backend  
mvnw spring-boot:run  

### 3. Chạy Frontend
cd QLCT-frontend  
npm install  
npm run dev
